                        trash rack is physical.

            screen wash set is physical.
            screen wash set consists of SW1 and SW2.
                        SW1 is physical.

                        SW2 is physical.
                        SW2 is instance of a ScrWshfull.
            screen wash set relates to waterbox.
            pump set is physical.
            pump set consists of pump1 and pump2.

                        pump1 is instance of a centrifugalPumpFull.
                        pump2 is physical.
                        pump2 is instance of a centrifugalPumpFull.
            pump set relates to screen wash set.
