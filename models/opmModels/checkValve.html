Fluid is physical.
Fluid exhibits Location and Flow direction.
            Location is physical.
            Location can be Leaking or Contained.
            Flow direction is physical.
            Flow direction can be Stopped or Moving.
Valve bodies is physical.
Valve bodies consists of End connection, Bonnet joint, Main body, Gasket, Cover plate, and Stud bolts.
            End connection is physical.
            Bonnet joint is physical.
            Main body is physical.
            Gasket is physical.
            Cover plate is physical.
Disc bodies is physical.
Disc bodies consists of Disc, Hinge pin, Seat ring, and Cotter pin.
            Disc is physical.
            Hinge pin is physical.
            Seat ring is physical.
            Cotter pin is physical.
Contain is physical.
Contain requires Valve bodies.
Contain changes Location from Leaking to Contained.
Flow control is physical.
Flow control requires Disc bodies.
Flow control changes Flow direction from Stopped to Moving.
