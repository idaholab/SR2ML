Feedwater is physical.
Feedwater exhibits Location and Temperature.
            Location is physical.
            Location can be Contained or Leaking.
            Temperature is physical.
            Temperature can be Low or High.
                        Low is initial.
                        High is final.
Feedwater housing is physical.
Feedwater housing consists of Tie spacers, Tie rods, Tube supports, and U-tubes.
            Tie spacers is physical.
            Tie rods is physical.
            Tube supports is physical.
            U-tubes is physical.
Drain is physical.
Drain exhibits Phase and Location.
            Phase is physical.
            Phase can be Steam or Liquid.
                        Steam is initial.
                        Liquid is final.
            Location is physical.
            Location can be Contained or Leaking.
Drain housing is physical.
Drain housing consists of Waterbox, Manhole access, Steam belt, and Standpipe isolators.
            Waterbox is physical.
            Manhole access is physical.
            Steam belt is physical.
            Standpipe isolators is physical.
Feedwater contain is physical.
Feedwater contain requires Feedwater housing.
Feedwater contain changes Location from Contained to Leaking.
Heat exchange is physical.
Heat exchange requires U-tubes.
Heat exchange changes Temperature from Low to High and Phase from Steam to Liquid.
Drain contain is physical.
Drain contain requires Drain housing.
Drain contain changes Location from Leaking to Contained.
