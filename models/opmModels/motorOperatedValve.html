Fluid is environmental and physical.
Fluid exhibits Flow and Location.
            Flow is physical.
            Flow can be Controlled by default or Uncontrolled.
            Location is physical.
            Location can be Contained by default or Leaking.
Valve assembly is physical.
Valve assembly exhibits Rotation.
            Rotation is physical.
            Rotation can be Rotating by default Stopped.
Valve assembly consists of Yoke, Disc, Stem nut, and Stem.
            Yoke is physical.
            Disc is physical.
            Stem nut is physical.
            Stem is physical.
Housing is physical.
Housing consists of Body, Bonnet, and Seal.
            Body is physical.
            Bonnet is physical.
            Seal is physical.
Actuator is physical.
Actuator exhibits Actuator rotation.
            Actuator rotation is physical.
            Actuator rotation can be Rotating by default Stop.
Actuator consists of Gear train and Motor.
            Gear train is physical.
            Gear train consists of Gear bearings, Drive sleeve, Torque switch, Spline, Worm wheel, and Worm.
                        Gear bearings is physical.
                        Drive sleeve is physical.
                        Torque switch is physical.
                        Spline is physical.
                        Worm wheel is physical.
                        Worm is physical.
            Motor is physical.
            Motor consists of Motor-gear coupling, Pump bearing, Rotor, and Stator.
                        Motor-gear coupling is physical.
                        Pump bearing is physical.
                        Rotor is physical.
                        Stator is physical.
AC power supply is environmental and physical.
Flow control is physical.
Flow control requires Valve assembly.
Flow control changes Flow from Uncontrolled to Controlled.
Contain is physical.
Contain requires Housing.
Contain changes Location from Leaking to Contained.
Rotation is physical.
Rotation requires Actuator.
Rotation changes Rotation from Stopped to Rotating.
AC powering is physical.
AC powering requires AC power supply.
AC powering changes Actuator rotation from Stop to Rotating.
